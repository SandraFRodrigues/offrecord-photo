# Offrecord_Photo

Plataforma web para fotógrafos com **galerias privadas**, **acesso por cliente** e **compra de fotografias**.

O objetivo do projeto é criar um produto real, focado em **arquitetura limpa**, **boas práticas em .NET** e **experiência de utilizador moderna**.

---

##  Funcionalidades

- Página pública com apresentação dos serviços
- Portfólio com álbuns públicos
- Galerias de fotografias por álbum
- Área do cliente (UI preparada para autenticação)
- Estados de fotografias (preview / locked)
- Interface em tema escuro
- Navegação clara e orientada a produto

> A autenticação e compra estão planeadas, mas ainda não implementadas (fase seguinte).

---

##  Arquitetura

O projeto segue uma separação clara de responsabilidades

### Princípios aplicados
- Clean Architecture
- Dependency Injection
- Separação de camadas
- Código simples e legível
- UI desacoplada do domínio

---

## Tecnologias

- ASP.NET Core (.NET)
- Razor Pages
- C#
- Bootstrap (base)
- CSS custom (tema escuro)
- In-memory repositories (para demonstração)
